# Fraud Detection System

## Problem
Fraud transactions are extremely rare (~0.17%) but have significant financial impact. Traditional rule-based systems fail to detect evolving fraud patterns.

## Approach
- Built classification model using XGBoost
- Focused on Recall to minimize false negatives
- Evaluated using Precision-Recall metrics due to data imbalance

## Results
- Detected 89% of fraudulent transactions
- Improved operational cost efficiency by ~24%

## Key Insight
Fraud transactions cluster in mid-range values (~$100–$200), indicating evasion strategies by fraudsters.

## Business Impact
Reduced financial loss and improved real-time fraud detection capability.
